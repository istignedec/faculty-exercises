API DOCUMENTATION

GENRE

all users :
/music/api/genres GET - returns a list of all genres
/music/api/genres/id GET - returns information of genre with specified id

ivan :
/music/api/genres POST - creates a new genre
/music/api/genres/id PUT - updates genre information (artists over their id)
/music/api/genres/id DELETE - deletes genre with specified id


ARTIST

all users :
/music/api/artists GET - returns a list of all artists
/music/api/artists/id GET - returns information of artist with specified id
/music/api/genres/id/artists GET - returns a lists of artists for specified genre id

ivan :
/music/api/artists POST - creates a new artist
/music/api/artists/id PUT - updates artist information
/music/api/artists/id DELETE - deletes artist with specified id


DOCUMENTATION

/music/api GET - returns a documentation page
